# The New York Times Adult Best Seller List - 2025
This static poster explored the books on the adult best seller list in
the year 2025. After noticing that most books I read claim that that are a "New York Times Best Seller",
I wanted to explore what this really means. The main question driving this inspection was, "How many books are New York Times
Best Sellers?" All data used was obtained through the PDF's located at https://www.hawes.com/pastlist.htm.

## Organization of Code
I began by writing a function that would convert the data provided in the PDF's into a single, clean, and ready to use data
frame which I named df_2025 then downloaded as a .csv file. I then moved onto the data visulizations. For each of the three
that I made, I included at the begining the code used to further subset my original data df_2025 to get different stories
out of my data. 

## Static Poster
After creating informative visuals, I put them together onto a poster to present them and my findings. Check it out
under *Final Data Vis Poster*!
